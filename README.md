# Proyecto: Estructura de Datos - Árboles

### Clasificación de Árboles
1. **Árbol General:** Nodos con número ilimitado de hijos.
2. **Árbol Binario:** Cada nodo tiene máximo 2 hijos.
3. **Árbol Binario de Búsqueda (ABB):** Mantiene un orden (menores a la izquierda, mayores a la derecha).
4. **Árbol AVL:** Árbol binario auto-balanceado mediante rotaciones.

### Justificación del Modelo
Se implementó un **Árbol Binario de Búsqueda (ABB)**. Se eligió este modelo porque permite demostrar de forma clara las reglas de jerarquía, relaciones de parentesco y métricas de profundidad además de facilitar operaciones de búsqueda y ordenamiento eficiente de datos.
